# PQuantomatic

Python Quantitative Analysis Library

## Installation

### From Source

```bash
git clone https://github.com/FriendlyInvader/PQuantomatic.git
cd PQuantomatic
pip install -e .
```

### For Development

```bash
pip install -e ".[dev]"
```

## Usage

```python
import pquantomatic

print(pquantomatic.__version__)
```

## Development

### Running Tests

```bash
pytest
```

### Installing Development Dependencies

```bash
pip install -e ".[dev]"
```
